# Sample Report

This file is a placeholder sample report for the repository.

After running `python harness/run_pack.py`, generated reports will appear in this directory with timestamped filenames.

Example output files:

- `report_20260306_153000.md`
- `report_20260306_171500.md`

These reports summarize:

- pass/fail counts
- category-level pass rates
- applied failure tags
- case-by-case evaluator notes
